# DGStuff

Ready to use self-hosted Digital Stuff Store in one binary file (or Docker).

## Installation

### Binary File
```bash
curl ...
```

### Docker
```bash
docker run ...
```

or Docker Compose:
```yml
services:
    dgstuff:
        image: agilistikmal/dgstuff:latest
        container_name: dgstuff
        ...
```
